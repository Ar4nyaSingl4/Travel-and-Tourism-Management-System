# ✈️ Travel and Tourism Management System

## 📌 About the Project

The **Travel and Tourism Management System** is a desktop-based application developed using **Java Swing**, **JDBC**, and **MySQL**. It automates travel agency operations by providing a centralized platform for user authentication, customer management, hotel and package booking, payment processing, invoice generation, and booking management. The application offers a user-friendly graphical interface and securely stores customer and booking information in a MySQL database.

## ✨ Features

- 🔐 User Login & Signup
- 🔑 Forgot Password Recovery
- 👤 Customer Management (Add, Update, View & Delete)
- 🏨 Hotel Booking
- 🎒 Travel Package Booking
- 📍 Destination Selection
- 💰 Automatic Booking Cost Calculation
- 📖 View Booking History
- ❌ Cancel Booking
- 💳 QR Code-Based Payment
- 🧾 Invoice Generation
- 🗄️ MySQL Database Integration
